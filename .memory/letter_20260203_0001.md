## 3. The "Pain" Log (CRITICAL)
* **Tried:** Using `asChild` pattern on Button component for Link integration
* **Failed:** React doesn't support this pattern natively without Radix UI
* **Workaround:** Created `getButtonClasses()` utility in `src/lib/button-styles.ts` and apply classes directly to Link components

* **Tried:** Using `errorMap` in Zod v4 enum validation
* **Failed:** Zod v4 changed API - `errorMap` no longer exists on enum
* **Workaround:** Changed to `message` property: `z.enum([...], { message: '...' })`

* **Tried:** Using `validationResult.error.errors` for Zod error iteration
* **Failed:** Zod v4 renamed property to `issues`
* **Workaround:** Changed to `validationResult.error.issues.forEach()`

* **Tried:** Google Fonts @import in globals.css after Tailwind import
* **Failed:** CSS warning about @import order
* **Workaround:** Removed CSS @import, using `next/font/google` in layout.tsx instead (more performant)
