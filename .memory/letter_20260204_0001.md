## 3. The "Pain" Log (CRITICAL)

* **Workaround:** Created `getButtonClasses()` in `src/lib/button-styles.ts`

* **Tried:** Using `errorMap` in Zod v4 enum
* **Failed:** Zod v4 API changed

* **Tried:** `validationResult.error.errors` for Zod errors
* **Failed:** Zod v4 renamed to `issues`
* **Workaround:** Use `validationResult.error.issues.forEach()`
